Mage Slider is a simple jQuery Slider for WordPress with Awesome Effects. You can use it for display you information nicely.

== Description ==

Mage Slider is an awesome jQuery Slider. 

Features:

* Shortcode Powred
* Very Lightweight
* Responsive Slider
* Work with every theme
* Unlimited Slides
* Open Source
* Developer Friendly
* Use Shortcode [mage-slider] for displaying the slider anywhere in your WordPress website.
* Live demo http://demo.wp-people.net/mage-slider/

How to use:
After successfully installed you will find a new menu name 'Slider' in your WordPress admin menu section, Clik on Add New under slider menu then you can add Your slider image, title and text. After save your all slide you can go Setting -> Mage Slider here you will find the slider settings, Choose you suitable setting and hit the save button. Now you are ready to disply the slider any where of your website such as Post/Page or Sidebar just use the shortcode [mage-slider].

== Installation ==

Upload the Mage FAQ plugin to your blog, Activate it, then use shortcode as your need).

1, 2, 3: You're done!
